# 💳 Explainable Fraud Detection using XGBoost

An end-to-end Machine Learning fraud detection system built using XGBoost, featuring advanced feature engineering, class imbalance handling (ADASYN), and Explainable AI techniques (SHAP & LIME) for transparent model predictions.

---

## 🚀 Project Overview

This project focuses on detecting fraudulent financial transactions using machine learning while ensuring model transparency and interpretability.

The system includes:
- Data preprocessing & feature engineering
- Handling class imbalance using ADASYN
- Model training using XGBoost
- Model evaluation & performance metrics
- Explainability using SHAP and LIME
- Streamlit-based deployment for interactive predictions

---

## 🛠 Tech Stack

- Python
- XGBoost
- Scikit-learn
- SHAP
- LIME
- Pandas & NumPy
- Streamlit

---

## 📊 Model Pipeline

1. Data Cleaning & Preprocessing  
2. Feature Engineering  
3. Class Imbalance Handling (ADASYN)  
4. Model Training (XGBoost)  
5. Model Evaluation  
6. Explainability (SHAP & LIME)  
7. Deployment using Streamlit  

---

## 📈 Explainable AI

This project integrates:

- **SHAP** for global feature importance and model interpretability.
- **LIME** for local explanation of individual predictions.

These tools ensure transparency in fraud detection decisions.
